# Untitled Cake Studios 🎮

A modern, advanced portfolio and game studio website built with pure HTML, CSS, and JavaScript.

## 🌟 Features

- **Modern Design**: Clean, professional dark theme with vibrant accent colors
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Beautiful fade-in animations and hover effects
- **Interactive Navigation**: Sticky navbar with smooth transitions
- **Game Showcase**: Dedicated section to display your game portfolio
- **Team Section**: Highlight team members with social links
- **Contact Form**: Functional contact form with validation
- **Safety Commitment**: Dedicated section emphasizing family-friendly values
- **Performance Optimized**: Smooth scrolling, parallax effects, and efficient animations

## 🚀 Quick Start

1. **Clone or download** this repository
2. **Open `index.html`** in your web browser
3. **Customize** the content to match your studio

## 🛠️ Customization

### Update Content
- Edit team member names and roles in `index.html`
- Add your game information in the Games section
- Update contact information and social links

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #ff6b6b;        /* Main accent color */
    --secondary: #4ecdc4;      /* Secondary accent */
    --dark-bg: #0f0f0f;        /* Background color */
    /* ... more variables ... */
}
```

### Add Real Games
Replace placeholder game cards with actual game information, screenshots, and links.
